## API

### Spacer Props

| Attribute | Type             | Description      | Default |
| --------- | ---------------- | ---------------- | ------- |
| x         | [Space](#spaces) | The x axis space | `1`     |
| y         | [Space](#spaces) | The y axis space | `1`     |
