# Grid

The layout `Grid` adapts to screen size and orientation, ensuring consistency across layouts.

```jsx
import { Grid } from '@nextui-org/react';
```

#### Default breakpoints

| Breakpoint  | <650px | ≥650px | ≥960px | ≥1280px | ≥1400px |
| ----------- | ------ | ------ | ------ | ------- | ------- |
| **default** | `100%` | 650px  | 960px  | 1280px  | 1400px  |
| **xs**      | `100%` | 650px  | 960px  | 1280px  | 1400px  |
| **sm**      | `100%` | `100%` | 960px  | 1280px  | 1400px  |
| **md**      | `100%` | `100%` | 960px  | 1280px  | 1400px  |
| **lg**      | `100%` | `100%` | `100%` | 1280px  | 1400px  |
| **xl**      | `100%` | `100%` | `100%` | `100%`  | 1400px  |
| **fluid**   | `100%` | `100%` | `100%` | `100%`  | `100%`  |

### APIs

<Spacer y={2} />

#### Grid Props

| Attribute        | Type                          | Accepted values                | Description                         | Default |
| ---------------- | ----------------------------- | ------------------------------ | ----------------------------------- | ------- |
| **xs**           | `boolean`                     | `true / false`                 | extra small devices (<650px)        | `false` |
| **sm**           | `boolean`                     | `true / false`                 | small devices (≥650px)              | `false` |
| **md**           | `boolean`                     | `true / false`                 | medium devices (≥960px)             | `false` |
| **lg**           | `boolean`                     | `true / false`                 | large devices (≥1280px)             | `false` |
| **xl**           | `boolean`                     | `true / false`                 | extra large devices (≥1400px)       | `false` |
| **direction**    | `Direction`                   | [Direction](#direction)        | CSS "flex-direction" property       | -       |
| **justify**      | `Justify`                     | [Justify](#justify)            | CSS "justify-content" property      | -       |
| **alignItems**   | `AlignItems`                  | [AlignItems](#align-items)     | CSS "align-items" property          | -       |
| **alignContent** | `AlignContent`                | [AlignContent](#align-content) | CSS "align-content" property        | -       |
| **css**          | `Stitches.CSS`                | -                              | Override Default CSS style          | -       |
| **as**           | `keyof JSX.IntrinsicElements` | -                              | Changes which tag component outputs | `div`   |
| ...              | `HTMLAttributes`              | `'id', 'className', ...`       | Native element props                | -       |

### Grid types

<Spacer y={1} />

#### Wrap

```ts
type Wrap = 'nowrap' | 'wrap' | 'wrap-reverse';
```

#### Display

```ts
type Display =
  | 'flex'
  | 'block'
  | 'grid'
  | 'inline'
  | 'inline-block'
  | 'inline-flex'
  | 'inline-grid';
```

#### Justify

```ts
type Justify =
  | 'flex-start'
  | 'center'
  | 'flex-end'
  | 'space-between'
  | 'space-around'
  | 'space-evenly';
```

#### Direction

```ts
type Direction = 'row' | 'row-reverse' | 'column' | 'column-reverse';
```

#### Align Items

```ts
type AlignItems = 'flex-start' | 'flex-end' | 'center' | 'stretch' | 'baseline';
```

#### Align Content

```ts
type AlignContent =
  | 'stretch'
  | 'center'
  | 'flex-start'
  | 'flex-end'
  | 'space-between'
  | 'space-around';
```
